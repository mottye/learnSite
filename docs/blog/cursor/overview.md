# Cursor とは

Cursor は、AI を組み込んだコードエディタです。VS Code をベースにしつつ、チャットやインライン補完で開発を助けてくれます。

---

## 主な機能

- **AI チャット**: コードの質問・リファクタ案・説明をその場で聞ける
- **インライン補完**: 書いている途中のコードを AI が続きを提案
- **コマンド**: カスタムコマンド（例: `/commit`）でワークフローを定型化
- **ルール**: プロジェクトごとのルール（`.cursor/rules/`）で AI の振る舞いを統一

---

## このブログでの使い方

このサイト（LearnSite）では、MkDocs の記事執筆や設定の変更に Cursor を使っています。

!!! tip "記事を書くとき"
    ルール（`.cursor/rules/mkdocs-learn-blog.mdc`）に従い、front matter の `description` と `tags` を付けると、SEO とタグ一覧の両方に効きます。
