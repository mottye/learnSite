# スタートガイド

はじめての方向けの入口です。ここから学びを始めましょう。

---

## このガイドでできること

1. **ブログの構成を理解する** — スタートガイド・ブログ・タグの役割
2. **記事の書き方** — Markdown で `docs/blog/` に追加するだけ
3. **ローカルで確認** — `mkdocs serve` でプレビュー

---

## クイックスタート

```bash
# 依存関係のインストール
pip install -r requirements.txt

# 開発サーバー起動（http://127.0.0.1:8000）
mkdocs serve
```

記事を追加するときは `docs/blog/` に `.md` ファイルを作成し、`mkdocs.yml` の `nav` の「ブログ」セクションにリンクを追加してください。
