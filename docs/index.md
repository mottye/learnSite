## 柱トピック（例）

カテゴリやシリーズを「柱」としてまとめると、読者が迷いにくくなります。

| トピック | 説明 |
|---------|------|
| [スタートガイド](guides/index.md) | 初心者向けの入口 |
| [ブログ](blog/index.md) | 記事一覧 |

必要に応じて `mkdocs.yml` の `nav` と `docs/` 内のフォルダで柱を増やしてください。
