# このサイトについて

**LearnSite** は、MkDocs + Material テーマで作った「学ぶためのブログ」です。

---

## 技術スタック

- [MkDocs](https://www.mkdocs.org/)
- [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
- Markdown で記事を管理

---

## サイトの構成

- **ホーム** — トップページ（入口・最新更新・柱トピック）
- **スタートガイド** — はじめての方向け
- **ブログ** — 記事一覧
- **タグ** — トピック別の索引

---

## 開発・公開

- ローカルプレビュー: `mkdocs serve`
- 静的サイト生成: `mkdocs build` → `site/` に出力
- GitHub Pages などに `site/` をデプロイして公開できます
